# Calculator-1C

Конфигурация 1С (обработка «Калькулятор»), выгруженная в исходные файлы и версионируемая через git.

## О проекте

Репозиторий содержит исходники конфигурации 1С:Предприятие, выгруженные в текстовом/XML-виде штатным механизмом платформы (**Конфигурация → Выгрузить конфигурацию в файлы...**). Такой формат позволяет вести историю изменений конфигурации в git и работает даже на базовых версиях платформы, где недоступно хранилище конфигурации.

## Структура репозитория

```
.
├── Configuration.xml          # корневое описание конфигурации
├── ConfigDumpInfo.xml         # служебная информация о выгрузке
├── CommonForms/
│   └── Калькулятор/
│       └── Ext/
│           ├── Form.xml       # описание формы
│           └── Form/
│               └── Module.bsl # код модуля формы
├── Languages/
│   └── Русский.xml            # язык интерфейса
└── Ext/
    └── HomePageWorkArea.xml   # настройка рабочей области начальной страницы
```

## Требования

- [OneScript](https://oscript.io) — для работы с пакетным менеджером `opm` и утилитами синхронизации
- Платформа 1С:Предприятие 8.3 (для открытия и редактирования конфигурации в Конфигураторе)

## Как обновить репозиторий после изменений в конфигурации

1. Откройте базу в режиме **Конфигуратор**.
2. Выполните **Конфигурация → Выгрузить конфигурацию в файлы...**, указав эту же папку — изменятся только те файлы, которые реально поменялись.
3. Зафиксируйте изменения в git:

```bash
git add .
git commit -m "Описание изменений"
git push
```

## Как развернуть конфигурацию из репозитория

1. Создайте новую информационную базу в 1С (или используйте существующую).
2. В Конфигураторе выполните **Конфигурация → Загрузить конфигурацию из файлов...**, указав путь к этой папке.
3. Обновите конфигурацию базы данных (**F7**).
